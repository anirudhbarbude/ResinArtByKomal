# **App Name**: Artistic Resin By Komal

## Core Features:

- Product Display: Showcase resin art products with images, descriptions, and prices. and redirect to instagram when people want to buy it
- Dynamic Layout: Implement a dynamic and visually appealing layout to enhance user experience.
- Instagram Redirection: Enable a 'Buy' button that redirects users to the linked Instagram account for purchase.
- Logo Integration: Display the 'AK' logo prominently on the website.
- Aesthetic Design: Implement a visually appealing and artistic design theme inspired by Komal's resin art.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to reflect the richness and depth often found in resin art.
- Background color: Light Lavender (#E6E6FA), a very low saturation tint of the primary color, creates a gentle backdrop that makes the artworks stand out without clashing.
- Accent color: Plum (#8E4585) for call to action buttons and interactive elements, enhancing visibility and engagement.
- Body and headline font: 'Belleza', a humanist sans-serif aligned to art and design; consider pairing it with 'Alegreya' serif if blocks of text longer than a short paragraph will appear in the app.
- Use elegant, handcrafted-style icons related to art, such as paintbrushes, palettes, and abstract shapes, to enhance visual appeal.
- Employ a fluid and responsive layout that adapts to different screen sizes, ensuring a seamless experience on all devices.
- Incorporate subtle animations, such as smooth transitions and hover effects, to create a dynamic and engaging user experience.
